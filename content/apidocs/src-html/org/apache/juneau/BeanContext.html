034/**
035 * Core class of the Juneau architecture.
036 * <p>
037 *    This class servers multiple purposes:
038 *    <ul class='spaced-list'>
039 *       <li>Provides the ability to wrap beans inside {@link Map} interfaces.
040 *       <li>Serves as a repository for metadata on POJOs, such as associated {@link BeanFilter beanFilters}, {@link PropertyNamer property namers}, etc...
041 *          which are used to tailor how POJOs are serialized and parsed.
042 *       <li>Serves as a common utility class for all {@link Serializer Serializers} and {@link Parser Parsers}
043 *             for serializing and parsing Java beans.
044 *    </ul>
045 *
046 *
047 * <h5 class='topic'>Bean Contexts</h5>
048 * <p>
049 *    Typically, it will be sufficient to use the existing {@link #DEFAULT} contexts for creating
050 *    bean maps.  However, if you want to tweak any of the settings on the context, you must
051 *    either clone the default context or create a new one from scratch (whichever is simpler for you).
052 *    You'll notice that this context class uses a fluent interface for defining settings.
053 * <p>
054 *    Bean contexts are created by {@link ContextFactory context factories}.
055 *    The settings on a bean context are fixed at the point they are created by the factory.
056 *
057 *
058 * <h5 class='topic'>BeanContext settings</h5>
059 *    <code>BeanContexts</code> have several settings that can be used to tweak behavior on how beans are handled.
060 *    These are denoted as the static <jsf>BEAN_*</jsf> fields on this class.
061 * <p>
062 *    Some settings (e.g. {@link BeanContext#BEAN_beansRequireDefaultConstructor}) are used to differentiate between bean and non-bean classes.
063 *    Attempting to create a bean map around one of these objects will throw a {@link BeanRuntimeException}.
064 *    The purpose for this behavior is so that the serializers can identify these non-bean classes and convert them to plain strings using the {@link Object#toString()} method.
065 * <p>
066 *    Some settings (e.g. {@link BeanContext#BEAN_beanFieldVisibility}) are used to determine what kinds of properties are detected on beans.
067 * <p>
068 *    Some settings (e.g. {@link BeanContext#BEAN_beanMapPutReturnsOldValue}) change the runtime behavior of bean maps.
069 * <p>
070 *    Settings are specified using the {@link ContextFactory#setProperty(String, Object)} method and related convenience methods.
071 *
072 * <h6 class='topic'>Examples</h6>
073 * <p class='bcode'>
074 *    <jc>// Construct a context from scratch.</jc>
075 *    BeanContext beanContext = ContextFactory.<jsm>create</jsm>()
076 *       .setProperty(BeanContext.<jsf>BEAN_beansRequireDefaultConstructor</jsf>, <jk>true</jk>)
077 *       .addNotBeanClasses(Foo.<jk>class</jk>)
078 *       .getBeanContext();
079 *
080 *    <jc>// Clone an existing context factory.</jc>
081 *    BeanContext beanContext = ContextFactory.<jsm>create</jsm>(otherConfig)
082 *       .setProperty(BeanContext.<jsf>BEAN_beansRequireDefaultConstructor</jsf>, <jk>true</jk>)
083 *       .addNotBeanClasses(Foo.<jk>class</jk>)
084 *       .getBeanContext();
085 * </p>
086 *
087 *
088 * <h5 class='topic'>Bean Maps</h5>
089 * <p>
090 *    {@link BeanMap BeanMaps} are wrappers around Java beans that allow properties to be retrieved and
091 *    set using the common {@link Map#put(Object,Object)} and {@link Map#get(Object)} methods.<br>
092 *    <br>
093 *    Bean maps are created in two ways...
094 *    <ol>
095 *       <li> {@link BeanContext#forBean(Object) BeanContext.forBean()} - Wraps an existing bean inside a {@code Map} wrapper.
096 *       <li> {@link BeanContext#newBeanMap(Class) BeanContext.newInstance()} - Create a new bean instance wrapped in a {@code Map} wrapper.
097 *    </ol>
098 *
099 *
100 * <h6 class='topic'>Examples</h6>
101 * <p class='bcode'>
102 *    <jc>// A sample bean class</jc>
103 *    <jk>public class</jk> Person {
104 *       <jk>public</jk> String getName();
105 *       <jk>public void</jk> setName(String name);
106 *       <jk>public int</jk> getAge();
107 *       <jk>public void</jk> setAge(<jk>int</jk> age);
108 *    }
109 *
110 *    <jc>// Wrap an existing bean in a new bean map</jc>
111 *    BeanMap&lt;Person&gt; m1 = BeanContext.<jsf>DEFAULT</jsf>.forBean(<jk>new</jk> Person());
112 *    m1.put(<js>"name"</js>, <js>"John Smith"</js>);
113 *    m1.put(<js>"age"</js>, 45);
114 *
115 *    <jc>// Create a new bean instance wrapped in a new bean map</jc>
116 *    BeanMap&lt;Person&gt; m2 = BeanContext.<jsf>DEFAULT</jsf>.newInstance(Person.<jk>class</jk>);
117 *    m2.put(<js>"name"</js>, <js>"John Smith"</js>);
118 *    m2.put(<js>"age"</js>, 45);
119 *    Person p = m2.getBean();  <jc>// Get the bean instance that was created.</jc>
120 * </p>
121 *
122 *
123 * <h5 class='topic'>Bean Annotations</h5>
124 * <p>
125 *    This package contains annotations that can be applied to
126 *    class definitions to override what properties are detected on a bean.
127 * <h6 class='topic'>Examples</h6>
128 * <p class='bcode'>
129 *    <jc>// Bean class definition where only property 'name' is detected.</jc>
130 *    <ja>&#64;Bean</ja>(properties={<js>"name"</js>})
131 *    <jk>public class</jk> Person {
132 *       <jk>public</jk> String getName();
133 *       <jk>public void</jk> setName(String name);
134 *       <jk>public int</jk> getAge();
135 *       <jk>public void</jk> setAge(<jk>int</jk> age);
136 *    }
137 * <p>
138 *    See {@link Bean @Bean} and {@link BeanProperty @BeanProperty} for more information.
139 *
140 *
141 * <h5 class='topic'>Beans with read-only properties</h5>
142 * <p>
143 *    Bean maps can also be defined on top of beans with read-only properties by adding a
144 *    {@link BeanConstructor @BeanConstructor} annotation to one of the constructors on the
145 *    bean class.  This will allow read-only properties to be set through constructor arguments.
146 * <p>
147 *    When the <code>@BeanConstructor</code> annotation is present, bean instantiation is delayed until the call to {@link BeanMap#getBean()}.
148 *    Until then, bean property values are stored in a local cache until <code>getBean()</code> is called.
149 *    Because of this additional caching step, parsing into read-only beans tends to be slower and use
150 *    more memory than parsing into beans with writable properties.
151 * <p>
152 *    Attempting to call {@link BeanMap#put(String,Object)} on a read-only property after calling {@link BeanMap#getBean()}
153 *    will result in a {@link BeanRuntimeException} being thrown.
154 *    Multiple calls to {@link BeanMap#getBean()} will return the same bean instance.
155 * <p>
156 *    Beans can be defined with a combination of read-only and read-write properties.
157 * <p>
158 *    See {@link BeanConstructor @BeanConstructor} for more information.
159 *
160 *
161 * <h5 class='topic'>BeanFilters and PojoSwaps</h5>
162 * <p>
163 *    {@link BeanFilter BeanFilters} and {@link PojoSwap PojoSwaps} are used to tailor how beans and POJOs are handled.<br>
164 *    <ol class='spaced-list'>
165 *       <li>{@link BeanFilter} - Allows you to tailor handling of bean classes.
166 *          This class can be considered a programmatic equivalent to the {@link Bean} annotation when
167 *          annotating classes are not possible (e.g. you don't have access to the source).
168 *          This includes specifying which properties are visible and the ability to programmatically override the execution of properties.
169 *       <li>{@link PojoSwap} - Allows you to swap out non-serializable objects with serializable replacements.
170 *    </ol>
171 * <p>
172 *    See {@link org.apache.juneau.transform} for more information.
173 *
174 *
175 * <h5 class='topic'>ClassMetas</h5>
176 * <p>
177 *    The {@link ClassMeta} class is a wrapper around {@link Class} object that provides cached information
178 *    about that class (e.g. whether it's a {@link Map} or {@link Collection} or bean).
179 * <p>
180 *    As a general rule, it's best to reuse bean contexts (and therefore serializers and parsers too)
181 *    whenever possible since it takes some time to populate the internal {@code ClassMeta} object cache.
182 *    By reusing bean contexts, the class type metadata only needs to be calculated once which significantly
183 *    improves performance.
184 * <p>
185 *    See {@link ClassMeta} for more information.
186 *
187 * @author Barry M. Caceres
188 * @author James Bognar (james.bognar@salesforce.com)
189 */

933   /**
934    * Construct a {@code ClassMeta} wrapper around a {@link Class} object.
935    *
936    * @param <T> The class type being wrapped.
937    * @param c The class being wrapped.
938    *    of type {@link Class} or {@link ClassMeta}.
939    * @return If the class is not an array, returns a cached {@link ClassMeta} object.
940    *    Otherwise, returns a new {@link ClassMeta} object every time.<br>
941    */
942   public <T> ClassMeta<T> getClassMeta(Class<T> c) {
943
944      // If this is an array, then we want it wrapped in an uncached ClassMeta object.
945      // Note that if it has a pojo transform, we still want to cache it so that
946      // we can cache something like byte[] with ByteArrayBase64Swap.
947      if (c.isArray() && findPojoSwap(c) == null)
948         return new ClassMeta(c, this);
949
950      // This can happen if we have transforms defined against String or Object.
951      if (cmCache == null)
952         return null;
953
954      ClassMeta<T> cm = cmCache.get(c);
955      if (cm == null) {
956
957         synchronized (this) {
958
959            // Make sure someone didn't already set it while this thread was blocked.
960            cm = cmCache.get(c);
961            if (cm == null) {
962
963               // Note:  Bean properties add the possibility that class reference loops exist.
964               // To handle this possibility, we create a set of pending ClassMetas, and
965               // call init (which finds the bean properties) after it's been added to the pending set.
966               for (ClassMeta pcm : pendingClassMetas)
967                  if (pcm.innerClass == c)
968                     return pcm;
969
970               cm = new ClassMeta<T>(c, this, true);
971               pendingClassMetas.addLast(cm);
972               try {
973                  cm.init();
974               } finally {
975                  pendingClassMetas.removeLast();
976               }
977               cmCache.put(c, cm);
978            }
979         }
980      }
981      return cm;
982   }

1092   ClassMeta getClassMeta(Type o, Map<Class<?>,Class<?>[]> typeVarImpls) {
1093      if (o == null)
1094         return null;
1095
1096      if (o instanceof ClassMeta)
1097         return (ClassMeta)o;
1098
1099      Class c = null;
1100      if (o instanceof Class) {
1101         c = (Class)o;
1102      } else if (o instanceof ParameterizedType) {
1103         // A parameter (e.g. <String>.
1104         c = (Class<?>)((ParameterizedType)o).getRawType();
1105      } else if (o instanceof GenericArrayType) {
1106         // An array parameter (e.g. <byte[]>.
1107         GenericArrayType gat = (GenericArrayType)o;
1108         Type gatct = gat.getGenericComponentType();
1109         if (gatct instanceof Class) {
1110            Class gatctc = (Class)gatct;
1111            c = Array.newInstance(gatctc, 0).getClass();
1112         } else if (gatct instanceof ParameterizedType) {
1113            Class gatctc = (Class<?>)((ParameterizedType)gatct).getRawType();
1114            c = Array.newInstance(gatctc, 0).getClass();
1115         } else {
1116            return null;
1117         }
1118      } else if (o instanceof TypeVariable) {
1119         if (typeVarImpls != null) {
1120            TypeVariable t = (TypeVariable) o;
1121            String varName = t.getName();
1122            int varIndex = -1;
1123            Class gc = (Class)t.getGenericDeclaration();
1124            TypeVariable[] tv = gc.getTypeParameters();
1125            for (int i = 0; i < tv.length; i++) {
1126               if (tv[i].getName().equals(varName)) {
1127                  varIndex = i;
1128               }
1129            }
1130            if (varIndex != -1) {
1131
1132               // If we couldn't find a type variable implementation, that means
1133               // the type was defined at runtime (e.g. Bean b = new Bean<Foo>();)
1134               // in which case the type is lost through erasure.
1135               // Assume java.lang.Object as the type.
1136               if (! typeVarImpls.containsKey(gc))
1137                  return object();
1138
1139               return getClassMeta(typeVarImpls.get(gc)[varIndex]);
1140            }
1141         }
1142         // We don't know the bounded type, so just resolve to Object.
1143         return object();
1144      } else {
1145         // This can happen when trying to resolve the "E getFirst()" method on LinkedList, whose type is a TypeVariable
1146         // These should just resolve to Object.
1147         return object();
1148      }
1149
1150      ClassMeta rawType = getClassMeta(c);
1151
1152      // If this is a Map or Collection, and the parameter types aren't part
1153      // of the class definition itself (e.g. class AddressBook extends List<Person>),
1154      // then we need to figure out the parameters.
1155      if (rawType.isMap() || rawType.isCollection()) {
1156         ClassMeta[] params = findParameters(o, c);
1157         if (params == null)
1158            return rawType;
1159         if (rawType.isMap()) {
1160            if (params.length != 2)
1161               return rawType;
1162            if (params[0].isObject() && params[1].isObject())
1163               return rawType;
1164            return new ClassMeta(rawType.innerClass, this).setKeyType(params[0]).setValueType(params[1]);
1165         }
1166         if (rawType.isCollection()) {
1167            if (params.length != 1)
1168               return rawType;
1169            if (params[0].isObject())
1170               return rawType;
1171            return new ClassMeta(rawType.innerClass, this).setElementType(params[0]);
1172         }
1173      }
1174
1175      return rawType;
1176   }

1193   ClassMeta[] findParameters(Type o, Class c) {
1194      if (o == null)
1195         o = c;
1196
1197      // Loop until we find a ParameterizedType
1198      if (! (o instanceof ParameterizedType)) {
1199         loop: do {
1200            o = c.getGenericSuperclass();
1201            if (o instanceof ParameterizedType)
1202               break loop;
1203            for (Type t : c.getGenericInterfaces()) {
1204               o = t;
1205               if (o instanceof ParameterizedType)
1206                  break loop;
1207            }
1208            c = c.getSuperclass();
1209         } while (c != null);
1210      }
1211
1212      if (o instanceof ParameterizedType) {
1213         ParameterizedType pt = (ParameterizedType)o;
1214         if (! pt.getRawType().equals(Enum.class)) {
1215            List<ClassMeta<?>> l = new LinkedList<ClassMeta<?>>();
1216            for (Type pt2 : pt.getActualTypeArguments()) {
1217               if (pt2 instanceof WildcardType || pt2 instanceof TypeVariable)
1218                  return null;
1219               l.add(getClassMeta(pt2, null));
1220            }
1221            if (l.isEmpty())
1222               return null;
1223            return l.toArray(new ClassMeta[l.size()]);
1224         }
1225      }
1226
1227      return null;
1228   }

1302   /**
1303    * Converts class name strings to ClassMeta objects.
1304    *
1305    * <dl>
1306    *    <dt>Example:</dt>
1307    *    <dd>
1308    * <ul>
1309    *    <li><js>"java.lang.String"</js>
1310    *    <li><js>"com.ibm.sample.MyBean[]"</js>
1311    *    <li><js>"java.util.HashMap<java.lang.String,java.lang.Integer>"</js>
1312    *    <li><js>"[Ljava.lang.String;"</js> (i.e. the value of <code>String[].<jk>class</jk>.getName()</code>)
1313    * </ul>
1314    *    </dd>
1315    * </dl>
1316    *
1317    * @param s The class name.
1318    * @return The ClassMeta corresponding to the class name string.
1319    */
1320   public ClassMeta<?> getClassMetaFromString(String s) {
1321      int d = 0;
1322      if (s == null || s.isEmpty())
1323         return null;
1324
1325      // Check for Class.getName() on array class types.
1326      if (s.charAt(0) == '[') {
1327         try {
1328            return getClassMeta(findClass(s));
1329         } catch (ClassNotFoundException e) {
1330            throw new RuntimeException(e);
1331         }
1332      }
1333
1334      int i1 = 0;
1335      int i2 = 0;
1336      int dim = 0;
1337      List<ClassMeta<?>> p = null;
1338      for (int i = 0; i < s.length(); i++) {
1339         char c = s.charAt(i);
1340         if (c == '<') {
1341            if (d == 0) {
1342               i1 = i;
1343               i2 = i+1;
1344               p = new LinkedList<ClassMeta<?>>();
1345            }
1346            d++;
1347         } else if (c == '>') {
1348            d--;
1349            if (d == 0 && p != null)
1350               p.add(getClassMetaFromString(s.substring(i2, i)));
1351         } else if (c == ',' && d == 1) {
1352            if (p != null)
1353               p.add(getClassMetaFromString(s.substring(i2, i)));
1354            i2 = i+1;
1355         }
1356         if (c == '[') {
1357            if (i1 == 0)
1358               i1 = i;
1359            dim++;
1360         }
1361      }
1362      if (i1 == 0)
1363         i1 = s.length();
1364      try {
1365         String name = s.substring(0, i1).trim();
1366         char x = name.charAt(0);
1367         Class<?> c = null;
1368         if (x >= 'b' && x <= 's') {
1369            if (x == 'b' && name.equals("boolean"))
1370               c = boolean.class;
1371            else if (x == 'b' && name.equals("byte"))
1372               c = byte.class;
1373            else if (x == 'c' && name.equals("char"))
1374               c = char.class;
1375            else if (x == 'd' && name.equals("double"))
1376               c = double.class;
1377            else if (x == 'i' && name.equals("int"))
1378               c = int.class;
1379            else if (x == 'l' && name.equals("long"))
1380               c = long.class;
1381            else if (x == 's' && name.equals("short"))
1382               c = short.class;
1383            else
1384               c = findClass(name);
1385         } else {
1386            c = findClass(name);
1387         }
1388
1389         ClassMeta<?> cm = getClassMeta(c);
1390
1391         if (p != null) {
1392            if (cm.isMap())
1393               cm = new ClassMeta(c, this).setKeyType(p.get(0)).setValueType(p.get(1));
1394            if (cm.isCollection())
1395               cm = new ClassMeta(c, this).setElementType(p.get(0));
1396         }
1397
1398         while (dim > 0) {
1399            cm = new ClassMeta(Array.newInstance(cm.getInnerClass(), 0).getClass(), this);
1400            dim--;
1401         }
1402
1403         return cm;
1404      } catch (ClassNotFoundException e) {
1405         throw new RuntimeException(e);
1406      }
1407   }

1704    * @param outer If class is a member class, this is the instance of the containing class.
1705    *    Should be <jk>null</jk> if not a member class.
1706    * @param value The value to convert.
1707    * @param type The class type to convert the value to.
1708    * @throws InvalidDataConversionException If the specified value cannot be converted to the specified type.
1709    * @return The converted value.
1710    */
1711   public <T> T convertToType(Object outer, Object value, ClassMeta<T> type) throws InvalidDataConversionException {
1712      if (type == null)
1713         type = (ClassMeta<T>)object();
1714
1715      try {
1716         // Handle the case of a null value.
1717         if (value == null) {
1718
1719            // If it's a primitive, then use the converters to get the default value for the primitive type.
1720            if (type.isPrimitive())
1721               return type.getPrimitiveDefault();
1722
1723            // Otherwise, just return null.
1724            return null;
1725         }
1726
1727         Class<T> tc = type.getInnerClass();
1728
1729         // If no conversion needed, then just return the value.
1730         // Don't include maps or collections, because child elements may need conversion.
1731         if (tc.isInstance(value))
1732            if (! ((type.isMap() && type.getValueType().isNotObject()) || (type.isCollection() && type.getElementType().isNotObject())))
1733               return (T)value;
1734
1735         if (tc == Class.class)
1736            return (T)(classLoader.loadClass(value.toString()));
1737
1738         if (type.getPojoSwap() != null) {
1739            PojoSwap f = type.getPojoSwap();
1740            Class<?> nc = f.getNormalClass(), fc = f.getSwapClass();
1741            if (isParentClass(nc, tc) && isParentClass(fc, value.getClass()))
1742               return (T)f.unswap(value, type, this);
1743         }
1744
1745         ClassMeta<?> vt = getClassMetaForObject(value);
1746         if (vt.getPojoSwap() != null) {
1747            PojoSwap f = vt.getPojoSwap();
1748            Class<?> nc = f.getNormalClass(), fc = f.getSwapClass();
1749            if (isParentClass(nc, vt.getInnerClass()) && isParentClass(fc, tc))
1750               return (T)f.swap(value, this);
1751         }
1752
1753         if (type.isPrimitive()) {
1754            if (value.toString().isEmpty())
1755               return type.getPrimitiveDefault();
1756
1757            if (type.isNumber()) {
1758               if (value instanceof Number) {
1759                  Number n = (Number)value;
1760                  if (tc == Integer.TYPE)
1761                     return (T)Integer.valueOf(n.intValue());
1762                  if (tc == Short.TYPE)
1763                     return (T)Short.valueOf(n.shortValue());
1764                  if (tc == Long.TYPE)
1765                     return (T)Long.valueOf(n.longValue());
1766                  if (tc == Float.TYPE)
1767                     return (T)Float.valueOf(n.floatValue());
1768                  if (tc == Double.TYPE)
1769                     return (T)Double.valueOf(n.doubleValue());
1770                  if (tc == Byte.TYPE)
1771                     return (T)Byte.valueOf(n.byteValue());
1772               } else {
1773                  String n = null;
1774                  if (value instanceof Boolean)
1775                     n = ((Boolean)value).booleanValue() ? "1" : "0";
1776                  else
1777                     n = value.toString();
1778                  if (tc == Integer.TYPE)
1779                     return (T)Integer.valueOf(n);
1780                  if (tc == Short.TYPE)
1781                     return (T)Short.valueOf(n);
1782                  if (tc == Long.TYPE)
1783                     return (T)Long.valueOf(n);
1784                  if (tc == Float.TYPE)
1785                     return (T)new Float(n);
1786                  if (tc == Double.TYPE)
1787                     return (T)new Double(n);
1788                  if (tc == Byte.TYPE)
1789                     return (T)Byte.valueOf(n);
1790               }
1791            } else if (type.isChar()) {
1792               String s = value.toString();
1793               return (T)Character.valueOf(s.length() == 0 ? 0 : s.charAt(0));
1794            } else if (type.isBoolean()) {
1795               if (value instanceof Number) {
1796                  int i = ((Number)value).intValue();
1797                  return (T)(i == 0 ? Boolean.FALSE : Boolean.TRUE);
1798               }
1799               return (T)Boolean.valueOf(value.toString());
1800            }
1801         }
1802
1803         if (type.isNumber()) {
1804            if (value instanceof Number) {
1805               Number n = (Number)value;
1806               if (tc == Integer.class)
1807                  return (T)Integer.valueOf(n.intValue());
1808               if (tc == Short.class)
1809                  return (T)Short.valueOf(n.shortValue());
1810               if (tc == Long.class)
1811                  return (T)Long.valueOf(n.longValue());
1812               if (tc == Float.class)
1813                  return (T)Float.valueOf(n.floatValue());
1814               if (tc == Double.class)
1815                  return (T)Double.valueOf(n.doubleValue());
1816               if (tc == Byte.class)
1817                  return (T)Byte.valueOf(n.byteValue());
1818               if (tc == Byte.class)
1819                  return (T)Byte.valueOf(n.byteValue());
1820               if (tc == AtomicInteger.class)
1821                  return (T)new AtomicInteger(n.intValue());
1822               if (tc == AtomicLong.class)
1823                  return (T)new AtomicLong(n.intValue());
1824            } else {
1825               if (value.toString().isEmpty())
1826                  return null;
1827               String n = null;
1828               if (value instanceof Boolean)
1829                  n = ((Boolean)value).booleanValue() ? "1" : "0";
1830               else
1831                  n = value.toString();
1832               if (tc == Integer.class)
1833                  return (T)Integer.valueOf(n);
1834               if (tc == Short.class)
1835                  return (T)Short.valueOf(n);
1836               if (tc == Long.class)
1837                  return (T)Long.valueOf(n);
1838               if (tc == Float.class)
1839                  return (T)new Float(n);
1840               if (tc == Double.class)
1841                  return (T)new Double(n);
1842               if (tc == Byte.class)
1843                  return (T)Byte.valueOf(n);
1844               if (tc == AtomicInteger.class)
1845                  return (T)new AtomicInteger(Integer.valueOf(n));
1846               if (tc == AtomicLong.class)
1847                  return (T)new AtomicLong(Long.valueOf(n));
1848            }
1849         }
1850
1851         if (type.isChar()) {
1852            String s = value.toString();
1853            return (T)Character.valueOf(s.length() == 0 ? 0 : s.charAt(0));
1854         }
1855
1856         // Handle setting of array properties
1857         if (type.isArray()) {
1858            if (vt.isCollection())
1859               return (T)toArray(type, (Collection)value);
1860            else if (vt.isArray())
1861               return (T)toArray(type, Arrays.asList((Object[])value));
1862            else if (StringUtils.startsWith(value.toString(), '['))
1863               return (T)toArray(type, new ObjectList(value.toString()).setBeanContext(this));
1864         }
1865
1866         // Target type is some sort of Map that needs to be converted.
1867         if (type.isMap()) {
1868            try {
1869               if (value instanceof Map) {
1870                  Map m = type.canCreateNewInstance(outer) ? (Map)type.newInstance(outer) : new ObjectMap(this);
1871                  ClassMeta keyType = type.getKeyType(), valueType = type.getValueType();
1872                  for (Map.Entry e : (Set<Map.Entry>)((Map)value).entrySet()) {
1873                     Object k = e.getKey();
1874                     if (keyType.isNotObject()) {
1875                        if (keyType.isString() && k.getClass() != Class.class)
1876                           k = k.toString();
1877                        else
1878                           k = convertToType(m, k, keyType);
1879                     }
1880                     Object v = e.getValue();
1881                     if (valueType.isNotObject())
1882                        v = convertToType(m, v, valueType);
1883                     m.put(k, v);
1884                  }
1885                  return (T)m;
1886               } else if (!type.canCreateNewInstanceFromString(outer)) {
1887                  ObjectMap m = new ObjectMap(value.toString(), defaultParser);
1888                  return convertToType(outer, m, type);
1889               }
1890            } catch (Exception e) {
1891               throw new InvalidDataConversionException(value.getClass(), type, e);
1892            }
1893         }
1894
1895         // Target type is some sort of Collection
1896         if (type.isCollection()) {
1897            try {
1898               Collection l = type.canCreateNewInstance(outer) ? (Collection)type.newInstance(outer) : new ObjectList(this);
1899               ClassMeta elementType = type.getElementType();
1900
1901               if (value.getClass().isArray())
1902                  for (Object o : (Object[])value)
1903                     l.add(elementType.isObject() ? o : convertToType(l, o, elementType));
1904               else if (value instanceof Collection)
1905                  for (Object o : (Collection)value)
1906                     l.add(elementType.isObject() ? o : convertToType(l, o, elementType));
1907               else if (value instanceof Map)
1908                  l.add(elementType.isObject() ? value : convertToType(l, value, elementType));
1909               else if (! value.toString().isEmpty())
1910                  throw new InvalidDataConversionException(value.getClass(), type, null);
1911               return (T)l;
1912            } catch (InvalidDataConversionException e) {
1913               throw e;
1914            } catch (Exception e) {
1915               throw new InvalidDataConversionException(value.getClass(), type, e);
1916            }
1917         }
1918
1919         if (type.isEnum()) {
1920            if (type.canCreateNewInstanceFromString(outer))
1921               return type.newInstanceFromString(outer, value.toString());
1922            return (T)Enum.valueOf((Class<? extends Enum>)tc, value.toString());
1923         }
1924
1925         if (type.isString()) {
1926            if (vt.isArray() || vt.isMap() || vt.isCollection() || vt.isBean()) {
1927               if (JsonSerializer.DEFAULT_LAX != null)
1928                  return (T)JsonSerializer.DEFAULT_LAX.serialize(value);
1929            } else if (vt.isClass()) {
1930               return (T)ClassUtils.getReadableClassName((Class<?>)value);
1931            }
1932            return (T)value.toString();
1933         }
1934
1935         if (type.isCharSequence()) {
1936            Class<?> c = value.getClass();
1937            if (c.isArray()) {
1938               if (c.getComponentType().isPrimitive()) {
1939                  ObjectList l = new ObjectList(this);
1940                  int size = Array.getLength(value);
1941                  for (int i = 0; i < size; i++)
1942                     l.add(Array.get(value, i));
1943                  value = l;
1944               }
1945               value = new ObjectList((Object[])value).setBeanContext(this);
1946            }
1947
1948            return type.newInstanceFromString(outer, value.toString());
1949         }
1950
1951         if (type.isBoolean()) {
1952            if (value instanceof Number)
1953               return (T)(Boolean.valueOf(((Number)value).intValue() != 0));
1954            return (T)Boolean.valueOf(value.toString());
1955         }
1956
1957         // It's a bean being initialized with a Map
1958         if (type.isBean() && value instanceof Map)
1959            return newBeanMap(tc).load((Map<?,?>) value).getBean();
1960
1961         if (type.canCreateNewInstanceFromObjectMap(outer) && value instanceof ObjectMap)
1962            return type.newInstanceFromObjectMap(outer, (ObjectMap)value);
1963
1964         if (type.canCreateNewInstanceFromNumber(outer) && value instanceof Number)
1965            return type.newInstanceFromNumber(outer, (Number)value);
1966
1967         if (type.canCreateNewInstanceFromString(outer))
1968            return type.newInstanceFromString(outer, value.toString());
1969
1970         if (type.isBean())
1971            return newBeanMap(type.getInnerClass()).load(value.toString()).getBean();
1972
1973      } catch (Exception e) {
1974         throw new InvalidDataConversionException(value, type, e);
1975      }
1976
1977      throw new InvalidDataConversionException(value, type, null);
1978   }

1980   /**
1981    * Converts the contents of the specified list into an array.
1982    * <p>
1983    *    Works on both object and primitive arrays.
1984    * <p>
1985    *    In the case of multi-dimensional arrays, the incoming list must
1986    *    contain elements of type n-1 dimension.  i.e. if {@code type} is <code><jk>int</jk>[][]</code>
1987    *    then {@code list} must have entries of type <code><jk>int</jk>[]</code>.
1988    *
1989    * @param type The type to convert to.  Must be an array type.
1990    * @param list The contents to populate the array with.
1991    * @return A new object or primitive array.
1992    */
1993   public Object toArray(ClassMeta<?> type, Collection<?> list) {
1994      if (list == null)
1995         return null;
1996      ClassMeta<?> componentType = type.getElementType();
1997      Object array = Array.newInstance(componentType.getInnerClass(), list.size());
1998      int i = 0;
1999      for (Object o : list) {
2000         if (! type.getInnerClass().isInstance(o)) {
2001            if (componentType.isArray() && o instanceof Collection)
2002               o = toArray(componentType, (Collection<?>)o);
2003            else if (o == null && componentType.isPrimitive())
2004               o = componentType.getPrimitiveDefault();
2005            else
2006               o = convertToType(null, o, componentType);
2007         }
2008         try {
2009            Array.set(array, i++, o);
2010         } catch (IllegalArgumentException e) {
2011            e.printStackTrace();
2012            throw e;
2013         }
2014      }
2015      return array;
2016   }
