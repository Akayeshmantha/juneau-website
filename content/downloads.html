Binaries
  • juneau-all-X.jar - Everything
  • juneau-samples-X.jar - Samples REST microservice (executable jar)
  • samples.cfg - Samples REST microservice config file
  • osgi-bundles/ - OSGi bundles
    • juneau-core-X.jar - Core libraries (serializers, parsers)
    • juneau-server-X.jar - REST servlet API
    • juneau-client-X.jar - REST client API
    • juneau-microservice-X.jar - Microservice API
Maven

Use the following Maven dependency to pull in the entire library into your project:

<dependency> <groupId>org.apache.juneau</groupId> <artifactId>juneau</artifactId> <version>6.0.0-incubating</version> </dependency>
