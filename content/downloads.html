The binaries consist of the following files:

  • juneau-all-X-incubating.jar - Single library containing all APIs.
  • juneau-samples-X-incubating.jar - Samples microservice. Run "java -jar juneau-samples-X-incubating.jar" to start up REST microservice samples on port 10000.
  • samples.cfg - Configuration file for samples microservice.
  • osgi-bundles/ - Juneau libraries as separate OSGi bundles.
    • juneau-core-X-incubating.jar - Core libraries (serializers, parsers).
    • juneau-server-X-incubating.jar - REST servlet API.
    • juneau-client-X-incubating.jar - REST client API.
    • juneau-microservice-X-incubating.jar - Microservice API.
